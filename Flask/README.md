# Start Flask Server

## MAC:

`python3 -m venv venc`

`. venv/bin/activate`

`pip install Flask`

`pip install flask_cors`

`python RestService.py`



## WINDOWS:

`py -3 -m venv venv`

`venv\Scripts\activate`

`pip install Flask`

`pip install flask_cors`

`python RestService.py`


Also see: https://flask.palletsprojects.com/en/2.0.x/installation/
