# Start Application

## Flask Back-End:

### Mac:

`cd Flask`

`python3 -m venv venv`

`. venv/bin/activate`

` pip install -r requirements.txt`

`python RestService.py`



### WINDOWS:

`py -3 -m venv venv`

`venv\Scripts\activate`

`pip install Flask`

`pip install flask_cors`

`python RestService.py`


Also see: https://flask.palletsprojects.com/en/2.0.x/installation/


## Front-End (Angular)

`cd front-end`

`npm install`

`ng serve --live-reload false`
